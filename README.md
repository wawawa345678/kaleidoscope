# ジェネラティブ万華鏡 (Generative Kaleidoscope)

マウスや指の動きから、対称的な万華鏡模様をリアルタイムに描くジェネラティブアート。
単一の HTML ファイルで動作し、ビルドや依存パッケージは不要です。

![Made with HTML5 Canvas](https://img.shields.io/badge/Made%20with-HTML5%20Canvas-7ee8fa)

## 使い方

`index.html` をブラウザで開くだけです。

```bash
# 例: そのままダブルクリックで開くか、簡易サーバーで配信
python -m http.server 8000
# → http://localhost:8000/index.html
```

キャンバス上をドラッグ（スマホは指でなぞる）すると描けます。

## 機能

- **対称の枚数** — 2〜24 枚の回転対称
- **筆の太さ** — 描画速度に応じて線が自然に細り、生き生きとしたラインに
- **色の流れる速さ** — 色相が時間とともに移り変わる
- **残像（フェード）** — 描いた軌跡を徐々に消して流れるような表現に
- **鏡** — 左右反転を加えた本格的な万華鏡モード
- **自動描画** — スピログラフ風のペンが自律的に模様を描く
- **画像を保存** — 現在のキャンバスを PNG として書き出し

## キーボードショートカット

| キー | 動作 |
|------|------|
| `H` | コントロールパネルの表示 / 非表示 |
| `Space` | 自動描画の開始 / 停止 |
| `C` | キャンバスをクリア |
| `S` | 画像を保存 |

## 動作環境

HTML5 Canvas に対応したモダンブラウザ（Chrome / Edge / Safari / Firefox）。
高 DPI ディスプレイやウィンドウのリサイズにも自動で対応します。
